# ContextDrop 核心概念

> 本文档定义项目核心术语，确保开发沟通一致。

## 领域概念

### 会话 (Session)
AI助手中的一次对话，包含完整的消息历史记录（用户提问 + AI回复）。
- 每个会话有唯一 ID
- 属于某个特定平台

### 消息 (Message)
会话中的单条对话记录，包含角色（用户/AI）和内容。

### 捕获 (Capture)
从 AI 平台页面解析会话的所有消息，构造结构化格式并保存到存储媒介。

### 批量捕获 (Batch Capture)
扫描某平台的所有会话列表，让用户选择后批量执行捕获。

### 上下文注入 (Context Injection)
将已捕获的会话上下文传递给目标处理器（如另一个 AI 助手），实现跨平台的上下文连续性。
